Best Seller History

Twenty Week History for a Book

Now that we have the ISBN of a book, let's look at some things we can do.

The NTY Books API also contains a history about best sellers, up to 20 weeks. You can search using many different parameters (which are listed in the documentation), or even with multiple such parameters by appending more than one to the query string. Once again, the results can be sorted using the sort-by and sort-order optional parameters.

Here, I will use the ISBN from the previous response (in particular, the primary_isbn13) to find historical best seller data for the book All the Light We Cannot See. Here is my request code:

var bestSellersUrl = 'http://api.nytimes.com/svc/books/v3/lists';

var url = bestSellersUrl + "/best-sellers/history";
url += "?isbn=9781476746586";
url += "&api-key=" + credentials.nytBooksKey;

request(url, function (err, response, body) {
	if (!err && response.statusCode < 400) {
		console.log(body);
	} else {
		//handle error condition
	}
});

Here, num_results is 1, which is expected, because searching by ISBN should only return one possible result. Therefore, the array results only has one object in it. Much of the data returned in this object is familiar from our previous request, so I will focus on the array inside this object that is called ranks_history.

The ranks_history array contains an object for each listing in a best sellers list for the past twenty weeks. Each time the book is on a best sellers list, you can see the name of that list (list_name and display_name), as well as when the list was published and created (published_date and bestsellers_date). Note that a book may appear on multiple best sellers lists in a single week, and can appear on the same list multiple weeks in a row. Using rank and weeks_on_list, you can gather data about how popular a book has been over the past twenty weeks.
